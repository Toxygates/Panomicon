This directory contains various scripts needed for Panomicon data generation and maintenance.
Many of the scripts depend on config.sh, which should be edited to set correct parameters prior to use.

Contents:

manager/ - scripts for working with t.Manager, the command-line data management tool
triplestore/ - scripts for manipulating data in a triplestore, such as fuseki
buildData/ - scripts for building a new database from scratch
rdf/ - scripts for accessing and manipulating various RDF data from external sources.
otg/ - scripts for maintaining Open TG-GATEs data
tcga/ - scripts for working with the Cancer Genome Atlas (TCGA), work in progress
experimental/ - various scripts under development
nibiohn/ - scripts for use inside NIBIOHN, which have not yet been adapted for public use
